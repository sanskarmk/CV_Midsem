Question 1 · 5 Marks

Sampling, Quantization & Contrast Stretching

📄 Original question

Q1(a) Explain the concept of "Sampling" and "Quantization" in the context of digital image processing. 2M

Q1(b) Given an image with pixel values ranging from 50 to 150, perform contrast stretching to expand the range to the full 8-bit range (0 to 255). What is the new value of a pixel with an original intensity of 100? 3M

Solution — Q1(a)

A continuous scene must be discretized in two independent ways to become a digital image:

SamplingQuantization
Digitizes the…spatial coordinates (x, y)amplitude / intensity
Decideshow many pixels (the grid size, spatial resolution)how many gray levels \(L=2^b\) (bit depth)
Analogywhere you place the measuring points along a scan linerounding each measured brightness to the nearest allowed level
Too little →blocky pixelationfalse contouring (banding)
Model answer

Sampling is digitizing the spatial coordinates of an image (choosing the pixel grid); quantization is digitizing the intensity values (choosing how many gray levels each pixel may take). Sampling fixes spatial resolution; quantization fixes intensity resolution. Together they convert the continuous function \(f(x,y)\) into a finite matrix of integers.

Solution — Q1(b)

Full-range linear contrast stretching maps \([r_{\min}, r_{\max}] \to [0, L-1]\):

\[ s = (r - r_{\min})\cdot\frac{L-1}{r_{\max}-r_{\min}} \]
Identify values
\(r_{\min}=50,\ r_{\max}=150,\ L-1=255,\ r=100\)
Slope
\(\dfrac{255}{150-50}=\dfrac{255}{100}=2.55\)
Apply
\(s=(100-50)\times 2.55 = 50\times 2.55 = 127.5\)
Round
\(s\approx \mathbf{128}\)
Answer

\(s = 127.5 \approx 128\). (A mid-range input maps to mid-range output — the hallmark of a linear stretch.)

Practice questions

Try each before revealing the worked answer.

P1.1 Contrast-stretch an image whose values span [30, 200] to full 8-bit range. Find the output for an input of 100. 3M
Slope
\(\dfrac{255}{200-30}=\dfrac{255}{170}=1.5\)
Apply to r=100
\(s=(100-30)\times 1.5 = 70\times 1.5 = 105\)
Answer

\(s = 105\).

P1.2 A grayscale image is 1024 × 768 at 8 bits/pixel. How much uncompressed storage does it need? What if it were 24-bit color? 2M
Bits
\(1024\times768\times8 = 6{,}291{,}456\) bits
Bytes / KB
\(\div 8 = 786{,}432\) bytes \(= 768\) KB (\(=0.75\) MB)
24-bit color
\(\times 3 = 2{,}359{,}296\) bytes \(= 2304\) KB \(= 2.25\) MB
Answer

768 KB grayscale; 2304 KB (2.25 MB) for 24-bit color. (General formula: bytes \(=M\cdot N\cdot b/8\).)

P1.3 After stretching [50, 150] → [0, 255], a pixel reads 153. What was its original intensity? 2M

Invert \(s=(r-50)\cdot2.55\) → \(r = \dfrac{s}{2.55}+50\).

Plug s=153
\(r = \dfrac{153}{2.55}+50 = 60 + 50 = 110\)
Answer

Original intensity \(= 110\).

P1.4 (i) How many gray levels does a 4-bit image have? (ii) How many bits are needed for 32 levels? (iii) Explain "false contouring". 2M
Answer

(i) \(L=2^4 = \mathbf{16}\) levels.  (ii) \(2^b=32 \Rightarrow b=\mathbf{5}\) bits.

(iii) False contouring is the banding that appears in smooth-gradient regions when too few quantization levels are used — the eye sees abrupt steps where the original had a continuous ramp. It's the visual symptom of insufficient intensity resolution.

P1.5 Distinguish spatial resolution from intensity resolution, and state what happens to a smooth gradient image if each is reduced. 2M
Answer

Spatial resolution = the smallest spatial detail that can be resolved, set by the number of samples/pixels (dpi). Reducing it makes the image pixelated / blocky; fine detail and edges become jagged.

Intensity resolution = the smallest distinguishable change in brightness, set by the number of gray levels \(L=2^b\). Reducing it causes false contouring — a smooth gradient breaks into visible bands.
