Question 5 · 5 Marks

Hough Spaces & Edge vs Corner

📄 Original question

Q5(a) In the Hough transform, determine the \(m\text{–}c\) and \(\rho\text{–}\theta\) space representation of the points \((2,0)\) and \((2,5)\). 3M

Q5(b) Differentiate between an edge and a corner. 2M

Solution — Q5(a)

① \(m\text{–}c\) space: each point → line \(c = -x\,m + y\)

Point (2, 0)
\(c = -2m + 0 \;\Rightarrow\; c = -2m\)
Point (2, 5)
\(c = -2m + 5\)
Try to intersect
\(-2m = -2m + 5 \Rightarrow 0 = 5\) — impossible. The two vote-lines are parallel (both slope −2), so they never meet.
▲ Why no intersection?

The line through \((2,0)\) and \((2,5)\) is the vertical line \(x=2\), which has infinite slope — it cannot be written as \(y=mx+c\). This is the exact failure case that motivates the polar form.

② \(\rho\text{–}\theta\) space: each point → sinusoid \(\rho = x\cos\theta + y\sin\theta\)

Point (2, 0)
\(\rho = 2\cos\theta + 0 = 2\cos\theta\)
Point (2, 5)
\(\rho = 2\cos\theta + 5\sin\theta\)
Intersect (set equal)
\(2\cos\theta = 2\cos\theta + 5\sin\theta \Rightarrow 5\sin\theta = 0 \Rightarrow \theta = 0^\circ\)
Find ρ at θ = 0°
\(\rho = 2\cos 0^\circ = 2\). So the common cell is \((\rho,\theta) = (2,\,0^\circ)\).
Answer

\(m\text{–}c\): lines \(c=-2m\) and \(c=-2m+5\) — parallel, no intersection (the line is vertical, slope ∞).

\(\rho\text{–}\theta\): sinusoids \(\rho=2\cos\theta\) and \(\rho=2\cos\theta+5\sin\theta\) intersect at \((\rho,\theta)=(2,\,0^\circ)\), representing the line \(x\cos0^\circ + y\sin0^\circ = 2\), i.e. \(\mathbf{x=2}\). The polar parameterisation succeeds precisely where \(m\text{–}c\) fails.

Solution — Q5(b) — edge vs corner

EdgeCorner
Intensity changes in…one direction (across the edge)two directions
Along the featureintensity is ~constantchanges every way you look
Structure-tensor eigenvaluesone large, one small \((\lambda_1\!\gg\!\lambda_2)\)both large \((\lambda_1,\lambda_2\) large\()\)
Harris response \(R\)\(R<0\)\(R>0\)
Localizationambiguous along the edge (aperture problem)well-localized in 2-D
Use for matchingweak (slides along itself)excellent — distinctive & repeatable
Model answer

An edge is a 1-D feature: intensity changes sharply in one direction but is constant along the edge, so you can't tell where you are along it (aperture problem) — its structure tensor has one large eigenvalue. A corner is a 2-D feature where two edges meet: intensity changes in all directions, both eigenvalues are large, so it is precisely localizable and highly distinctive — which is why corners (not edges) are used as matching keypoints.

Practice questions

P5.1 Find the \(m\text{–}c\) and \(\rho\text{–}\theta\) representation of the line through \((1,1)\) and \((3,3)\). 3M
m–c lines
\((1,1)\!: c=-m+1;\;\;(3,3)\!: c=-3m+3\)
Intersect
\(-m+1=-3m+3 \Rightarrow 2m=2 \Rightarrow m=1,\ c=0\) → line \(y=x\)
ρ–θ
\(\rho=\cos\theta+\sin\theta\) and \(\rho=3\cos\theta+3\sin\theta\); equal ⇒ \(\cos\theta+\sin\theta=0 \Rightarrow \tan\theta=-1 \Rightarrow \theta=135^\circ\)
ρ at 135°
\(\rho=\cos135^\circ+\sin135^\circ = -\tfrac{1}{\sqrt2}+\tfrac{1}{\sqrt2}=0\)
Answer

\(m\text{–}c\): \((m,c)=(1,0)\) → \(y=x\).   \(\rho\text{–}\theta\): \((\rho,\theta)=(0,135^\circ)\) — consistent (\(x\cos135^\circ+y\sin135^\circ=0 \Rightarrow y=x\)).

P5.2 Points \((1,3)\) and \((4,3)\) — a horizontal line. Give both representations. 3M
m–c
\(c=-m+3\) and \(c=-4m+3\); intersect ⇒ \(3m=0 \Rightarrow m=0,\ c=3\) → \(y=3\)
ρ–θ
\(\rho=\cos\theta+3\sin\theta\) and \(\rho=4\cos\theta+3\sin\theta\); equal ⇒ \(3\cos\theta=0 \Rightarrow \theta=90^\circ\)
ρ at 90°
\(\rho=\cos90^\circ+3\sin90^\circ = 0+3 = 3\)
Answer

\((m,c)=(0,3)\) → \(y=3\);  \((\rho,\theta)=(3,90^\circ)\). Here \(m\text{–}c\) works fine because a horizontal line has finite slope (0).

P5.3 Points \((4,1)\) and \((4,6)\). Show that \(m\text{–}c\) fails and find the \(\rho\text{–}\theta\) line. 2M
m–c
\(c=-4m+1\) and \(c=-4m+6\) — parallel (slope −4), no intersection → vertical line, slope ∞.
ρ–θ
\(\rho=4\cos\theta+\sin\theta\) and \(\rho=4\cos\theta+6\sin\theta\); equal ⇒ \(5\sin\theta=0 \Rightarrow \theta=0^\circ,\ \rho=4\)
Answer

\(m\text{–}c\) fails (parallel). \(\rho\text{–}\theta=(4,0^\circ)\) → line \(x=4\).

P5.4 State the single biggest advantage of the \(\rho\text{–}\theta\) parameterisation over \(m\text{–}c\). 1M
Answer

Both parameters are bounded (\(\theta\in[0,180^\circ)\), \(\rho\in[-D,D]\)), so it can represent every line including vertical ones in a finite accumulator — unlike \(m\text{–}c\), where vertical lines need \(m\to\infty\).

P5.5 Explain the "aperture problem" and why it makes edges poor matching features. 2M
Answer

Viewing an edge through a small window, you can detect motion across the edge but not along it — sliding the window along the edge looks identical. So an edge point can't be uniquely matched between images (it could correspond to many points along the same edge). A corner removes this ambiguity because shifting in any direction changes the appearance, giving a unique, repeatable match.
